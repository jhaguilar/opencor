The following issues/limitations have been reported/observed while using OpenCOR on our supported platforms. So, they may apply to you or not, depending on the platform you use.

Windows, Linux and OS X
  • By default, OpenCOR uses your system's language for menus, dialog boxes, etc., as long as it is either English or French (please contact us if you would like OpenCOR to support another language). If your system uses another language, then OpenCOR will default to English. Otherwise, if you specifically specify English or French, then please be aware that system messages, dialog boxes, etc. will still be displayed in your system's language.
Windows
  • OpenCOR's File Browser plugin may, on some systems, result in OpenCOR being slow to respond at startup. This has nothing to do with OpenCOR, but with Windows or, more likely, a program which tightly integrates with it. There are several known causes and solutions to this problem. However, should none of those solutions work for you, you might then have to uninstall the plugin.

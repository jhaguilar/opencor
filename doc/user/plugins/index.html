OpenCOR is a plugin-based application. This means that if no plugins are enabled, then OpenCOR can do next to nothing. As can be seen by opening the Plugins dialog box (by selecting the Tools | Plugins... menu or by clicking here) and unselecting Show only selectable plugins (if necessary), there are different types of plugins:

  • Organisation
  • Editing
  • Simulation
  • Miscellaneous
  • API
  • Third-party

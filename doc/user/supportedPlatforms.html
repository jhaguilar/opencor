OpenCOR can be used on Windows, Linux and OS X.

Windows

OpenCOR is supported on Windows 7 (both the 32-bit and 64-bit versions), but it should also work on Windows Vista and Windows XP SP3.

Linux

OpenCOR is supported on Ubuntu 12.04 LTS (Precise Pangolin; both the 32-bit and 64-bit versions), but it might also work with other versions of Ubuntu, as well as with other Linux distributions (though OpenCOR might need additional libraries to work).

OS X

OpenCOR is supported on OS X 10.8 (Mountain Lion).

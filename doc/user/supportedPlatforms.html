OpenCOR can be used on Windows, Linux and OS X.

Windows

OpenCOR is supported on Windows 7 (32-bit and 64-bit), but it should also work on Windows XP SP2 and Windows Vista.

Linux

OpenCOR is supported on Ubuntu 12.04 LTS (Precise Pangolin; 32-bit and 64-bit) and 12.10 (Quantal Quetzal; 32-bit and 64-bit), but it might also work with other versions of Ubuntu. It might also work with other Linux distributions, but this might require additional libraries.

OS X

OpenCOR is supported on OS X 10.8 (Mountain Lion).

OpenCOR offers two different types of interfaces:

  • Command line interface
  • Graphical user interface

Please note that, so far, limited work has been done on the command line interface. We have merely demonstrated that OpenCOR can be used either as a command line tool or as a graphical tool on our supported platforms.

The command line version of OpenCOR is therefore very limited and will, in fact, remain so compared to the graphical version. However, its use will make it easier and more convenient to handle certain tasks (e.g. to run several SED-ML files, to export a CellML file to C++).

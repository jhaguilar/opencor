OpenCOR offers a consistent graphical user interface across its supported platforms. The look and feel of the interface is mostly determined by the plugins you have selected. The first time you run OpenCOR, it will look something like this:

This is because some plugins have been selected for you by default. Therefore, to unselect all the plugins will result in OpenCOR looking empty:

Only some basic functionalities are available from the menu:

  • File:
    • Exit ~ Alt+F4: exit OpenCOR.
  • View:
    • Status Bar: show/hide the status bar.
    • Full Screen ~ F11: switch to / back from full screen mode.
  • Tools:
    • Language: select the language to be used by OpenCOR. Currently, this can either be your system's language, English or French.
    • Plugins...: select plugins.
    • Reset All: reset all your settings.
  • Help:
    • Home Page: open the OpenCOR home page.
    • About...: some general information about OpenCOR.

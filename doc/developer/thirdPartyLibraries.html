Third-party Libraries

OpenCOR uses external APIs, as well as software- and image-based third-party libraries, all of which are listed below, together with the type of license under which they are released. In all cases, only the files actually required by OpenCOR are in our Git repository.

APIs

  • CellML API 1.11: an interface to manipulate and process CellML documents.
    • Location: [OpenCOR]/src/plugins/api/CellMLAPI.
    • License: MPL v1.1, GPL v2.0 and LGPL v2.1.

Software

  • Google Code Prettify: a JavaScript module and CSS file for syntax highlighting of source code snippets in an HTML page.
    • Location: [OpenCOR]/doc/3rdparty/googleCodePrettify.
    • Licenses: Apache v2.0.
  • LibQxt 0.6.2: an extension library for Qt providing a suite of cross-platform utility classes to add functionality not readily available in the Qt toolkit.
    • Location: [OpenCOR]/src/3rdparty/LibQxt.
    • Licenses: CPL v1.0 and LGPL v2.1.
  • LLVM 3.1: a collection of modular and reusable compiler and toolchain technologies.
    • Location: [OpenCOR]/src/plugins/3rdparty/LLVM.
    • License: UoI/NCSA.
  • SUNDIALS 2.5.0: a SUite of Non-linear and DIfferential/ALgebraic equation Solvers of which we use CVODE and IDA.
    • Location: [OpenCOR]/src/plugins/3rdparty/SUNDIALS.
    • License: BSD.
  • QJson 0.7.1: a library that maps JSON data to QVariant objects.
    • Location: [OpenCOR]/src/plugins/3rdparty/QJson.
    • License: LGPL v2.1.
  • QScintilla 2.6.2: a Qt port of the Scintilla C++ editor control.
    • Location: [OpenCOR]/src/plugins/3rdparty/QScintilla.
    • License: GPL v2.0, GPL v3.0 and commercial.
  • QtMmlWidget 2.4.1: a class which renders mathematical formulas written in MathML 2.0.
    • Location: [OpenCOR]/src/plugins/3rdparty/QtMmlWidget.
    • Licenses: LGPL v2.1.
  • QtSingleApplication: a class which ensures that only one instance of OpenCOR can be run at once.
    • Location: [OpenCOR]/src/3rdparty/QtSingleApplication.
    • License: LGPL v2.1.
    • Note: we use the version used by Qt Creator and which can be found under [QtCreator]/src/shared/qtsingleapplication and [QtCreator]/src/shared/qtlockedfile.
  • Qwt 6.0.1: a set of widgets and utility classes primarily aimed at programs with a technical background.
    • Location: [OpenCOR]/src/plugins/3rdparty/Qwt.
    • License: Qwt License v1.0 (a slightly less restricive version of LGPL v2.1) and LGPL v2.1 for the MathML text engine which is based on some of QtMmlWidget's code.

Images

  • Printable world flags: the two (British and French) flag icons used in the menu to specify the locale to be used by OpenCOR.
    • Location: [OpenCOR]/res/flags.
    • License: none.
  • Boomy multiplatform: the main application icon.
    • Location: [OpenCOR]/res/boomy.
    • License: none.
  • Oxygen 4.9.0: the remaining icons used in OpenCOR.
    • Location: e.g. [OpenCOR]/res/oxygen.
    • License: LGPL v3.0.
